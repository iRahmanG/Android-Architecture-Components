# Android Architecture Components 🚀

This repository contains learning material, code examples, and projects based on the **"Android Architecture Components and Jetpack in Hindi"** YouTube series by [Cheezy Code](https://youtube.com/playlist?list=PLRKyZvuMYSIO0jLgj8g6sADnD0IBaWaw2&si=FDvyVij3HUlu8SVR).  
The purpose of this repo is to track my progress and consolidate the concepts covered in the playlist, including **MVVM**, **LiveData**, **ViewModel**, **Room**, and other essential Jetpack components.

- Jetpack Overview
